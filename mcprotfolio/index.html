my work

Here are some of the projects I've worked on recently. Feel free to explore:

    Denizen

    A scripting language and engine for Paper servers, allowing both quick and easy customization (due to functions like hot reloading) and creation of large scale projects. Due to the project's scale, this has made me familiar with many areas of Minecraft and its codebase.

    Clientizen

    A client-side mod written using the Fabric API and mod loader. Allows receving scripts and sets of customizations from the server as DenizenScript scripts, and applying them client-side. This allows custom GUIs, full control of client-side features, advanced text and rendering detections, and much more, opening up previously inaccessible modding options.

    Paper

    Paper is the biggest and most widely used server software for Minecraft (your server is probably using it/a fork of it right now!). I have made several contributions to Paper over the years, showing my understanding of both plugin development and the API itself.

    Adventure

    Adventure is a set of APIs for UI and backend Minecraft development, supporting everything from text components to raw NBT data. It's implemented by many of the biggest Minecraft server APIs, and I have made several contributions there as well, mainly to the NBT and text component APIs.

    NMS

    NMS (net.minecraft.server) refers to a Minecraft server's internals, I.e. Mojang's code. It is often used for more advanced modding functionality that isn't provided by traditional APIs. I have had extensive experience working with many areas of NMS over the years, see some examples below:

    NBT

    NBT is the file & data format used by Minecraft to save various forms of data, and is often used for certain advanced forms of modding, reading offline data, and certain optimizations. I have worked with NBT many times, giving me lots of familiarity with the format and relevant libraries.

    Packets

    Packets are the actual bits of data sent back and forth by the Minecraft client and server. Manipulating them is often done for more finite control over data the server wouldn't normally accept, optimizations that may come from modding at the packet level, or intercepting certain events that wouldn't be detectable otherwise.

    Reflection

    "Reflection" refers to a set of Java APIs used to dynamically access internal Java constructs (fields, methods, classes, etc.) at runtime. It's often used in Minecraft modding to write more dynamic code, access parts of NMS that are normally restricted, and much much more.
